# Автотест для проверки футера сайта only.digital

Этот проект содержит автоматизированный тест на Python, написанный с использованием фреймворков Selenium и Pytest.

Тест предназначен для проверки наличия и корректности ключевых элементов в футере (подвале) сайта `https://only.digital/`.

## Требования

*   Python 3.8+
*   Браузер Google Chrome

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/SanMog/only-digital-qa-test/tree/main
    cd only_digital_autotest
    ```

2.  **Создайте и активируйте виртуальное окружение:**

    *   Для Windows:
        ```bash
        python -m venv venv
        .\venv\Scripts\activate
        ```
    *   Для macOS/Linux:
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Установите необходимые зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    *Примечание: Selenium 4+ автоматически управляет `chromedriver`, поэтому его ручная установка не требуется.*

4.  **Запустите тест:**
    Выполните команду из корневой папки проекта:
    ```bash
    pytest
    ```

## Что проверяет тест

Скрипт `tests/test_footer.py` выполняет следующие шаги:
1.  Открывает главную страницу сайта.
2.  Убеждается, что элемент `<footer>` присутствует и виден на странице.
3.  Проверяет корректность текста с копирайтом (© 2014 - 2025).
4.  Находит все ссылки на социальные сети (Behance, Dprofile, Telegram, VK) и проверяет, что они видимы и кликабельны.
5.  Проверяет, что ссылка "Политика конфиденциальности" ведет на корректный URL.````
